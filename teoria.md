
## Single Page Application

Una Single Page Application (SPA) es un tipo de aplicación web que carga una única página HTML y, a partir de ahí, actualiza dinámicamente el contenido mediante interacciones del usuario. En lugar de cargar páginas completas desde el servidor, una SPA utiliza tecnologías como AJAX para recuperar y renderizar datos de manera asincrónica, proporcionando una experiencia de usuario más fluida. Al minimizar las recargas de página, las SPAs optimizan el rendimiento al reducir la carga en el servidor y permiten una navegación más rápida. Frameworks populares como React, Angular y Vue.js son comúnmente utilizados para desarrollar SPAs. Este enfoque mejora la interactividad y la eficiencia, brindando una experiencia de usuario más cercana a la de aplicaciones nativas.

## JSX

JSX (JavaScript XML) es una extensión de JavaScript utilizada comúnmente con bibliotecas como React para describir la interfaz de usuario de manera declarativa. En lugar de escribir código JavaScript para manipular el DOM de manera imperativa, JSX permite la creación de elementos UI utilizando una sintaxis similar a XML/HTML dentro del código JavaScript. Aunque parece un lenguaje de marcado, JSX se compila a llamadas de funciones JavaScript y se integra con la lógica de la aplicación. Facilita la creación de componentes reutilizables y mejora la legibilidad del código al proporcionar una representación visual de la jerarquía de la interfaz de usuario. JSX puede contener expresiones JavaScript en llaves {} para evaluar dinámicamente valores. Su adopción es común en el desarrollo de aplicaciones web con React.

## Estructura de un proyecto React hecho con Vite

### carpetas
- node_modules 
Es una carpeta donde estan todas las dependencias que se usan y se  van a usar de node js
- public
Aqui es donde se van a meter elementos estaticos y recursos que se vayan a utilizar en el proyecto, ejemplo: imagenes, videos
- src
En esta carpeta es donde vamos a trabajar y vamos a poner todo nuestro codigo
- src/assert
En esta subcarpeta que se encuentra dentro de src ubicaremos todos los recursos que vayamos a usar en el codigo ejemplo:imagenes,videos
